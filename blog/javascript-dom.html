Javascript Fundamentals

Sprint 3, May 2022

Basically HTML, CSS and Javascript are the three fundamentals that allow interactive websites found all over the world wide web to exist, to be created and changed on-going. For context I am running through what is HTML, CSS and JavaScript.

HTML, stands for hypertext markup language and it is the building block for the internet.

Webpages from websites based all over the world wide web (www) use HTML syntax to create structure for their content. A webpages actual content would likely contain text and images, maybe video or audio files, it would likely link to other webpages from the same website or perhaps an external site or two. HTML syntax gives us the ability to build a framework for a webpage and structure our sites content in a way which is user friendly for the target audience of the webpage or project.

Imagine a house, 3 bedrooms home with a family bathroom and two stories. That is the HTML of the building. But at this stage it's a blueprint on the desk of a project manager, a plan for a structure with content inbetween.

HTML is enhanced and modified by other technologies, one of them is CSS, which stands for cascading styles sheets.

CSS is used to control the aesthetic, the formatting and layout of webpages. CSS describes how the HTML elements, the framework and the content (text, images etc) should be presented on the webpage.

Fast forward 6 months to now, that house blueprint has come to life and the house is built. Made of red bricks with a solid wood front-door, which can be seen from the street. Inside each room of the family home there is a coorindiating collection of attributes(rules for how it should look). The attributes modify the HTML framework and the content it contains (ie. carpetted floors, wooden stairs and painted walls, drapes framming the windows). As a consequence of the hardwork done by HTML + CSS the user is left feeling like they are at home.

The other technology that commonly modifies HTML is JS or Javascript in fact 97% of websites on the web use JavaScript according to Wikipedia.

JavaScript, is an object orientated programming language, unlike the HTML and CSS which although important, aren't powerful. JS determines the dynamic and interactive elements on the page and brings life to a page, from static to dynamic and responsive. JS determines what happens when users click, hover or type within certain elements, makes them interactive. This is kind of like the functionality of the building.

Back to the house, you know the layout, structure and what it looks like, the aesthetic. But what can you do within the house? You can eat… sleep… open doors... turn the tap on/off... make meals… plug the tv in and watch Netflix... anything that you do at home, really! That is the JavaScript of the building. It brings the capability and function of the home alive for the user.

What is Control Flow?

By default JS code is executed from top to bottom, line by line however we can change this Control Flow. This allows our program to make decisions about what code is executed and when. Control flow is all about the order in which our code is run. We are able to control the flow of our code and either execut or loop over certain sections of code depending if a given condition is true or false. This gives our program the ability to change based on conditions and intelligently automate repeative tasks.

Conditional Statements:

Conditional statements are basically checks to see if a certain consition is either truw or false. If the consition is true then run code A, if it's false then run code B.

What are Loops?

Loops, are a conditional statement that allow us to perform repetitive tasks with less code. For example using loops we can iterate over an array and perform some sort of action on each piece of data in the array, or we can use a loop to compile a list of all the items.

There are a variety of Loop statement, such as 'for' loop, 'do/while' loop, 'for..in' loop and finally 'for..of' loop.
